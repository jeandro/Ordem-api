# Sobre o projeto

Desenvolver Back-End Java para geração de Ordem de serviço com rotinas de pesquisa e 
Manutenção dos dados.


# Tecnologias utilizadas
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- Junit
- Swagger
- flyway
- Postgresql
- Lombok


# Como executar o projeto

## Back end
Pré-requisitos:

- Java 1.8

- PostgreSQL 9.6

## Instalação  

Banco de Dados PostgreSQL 9.6
disponivel na url
